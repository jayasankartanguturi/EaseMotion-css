Utility Classes

Seven Ways to Reveal

Scroll down to watch each element animate into view exactly once. Mix and match with stagger delays for expressive sequences.

🌫️ .ease-reveal-fade

Fade

Smoothly fades in with a subtle scale, giving a clean, minimal entrance effect.

⬆️ .ease-reveal-up

Slide Up

Reveals from below — the element rises upward while fading in from the bottom.

⬇️ .ease-reveal-down

Slide Down

Reveals from above — drops downward while fading in from the top of the page.

⬅️ .ease-reveal-left

Slide Left

Sweeps in from the left — horizontal translation with opacity for a dynamic feel.

➡️ .ease-reveal-right

Slide Right

Sweeps in from the right — pairs perfectly with ease-reveal-left for mirrored effects.

🔍 .ease-reveal-zoom

Zoom

Scales up from 80 % to full size with a spring easing for a satisfying pop effect.

🌀 .ease-reveal-rotate

Rotate

Rotates in from −6° while fading, adding energy and personality to content blocks.

Built for Performance

Only opacity and transform are animated — the two GPU-composited properties. No layout recalculations, no jank.

Elements start hidden via CSS and receive .ease-reveal-active when the Intersection Observer fires. The observer is immediately unregistered afterwards to keep memory usage near zero.

View scroll-reveal.js ↗ 
<!-- Add a class — that's it -->
<div class="ease-reveal-up">
  Reveals from below
</div>

<!-- Combine with stagger delays -->
<div class="ease-reveal-fade ease-reveal-delay-1">
  First item
</div>
<div class="ease-reveal-fade ease-reveal-delay-2">
  Second item
</div>
<div class="ease-reveal-fade ease-reveal-delay-3">
  Third item
</div>

<!-- Load the observer once -->
<script src="core/scroll-reveal.js"></script>
